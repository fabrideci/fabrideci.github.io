# fabrideci.github.io

My personal site — IoT Infrastructure Architect & Lead DevOps Engineer.

**→ Live at [fabriziodecicco.dev](https://fabriziodecicco.dev)**

One static `index.html`: no framework, no build step, no dependencies, no
trackers. Auto light/dark, accessible, with a small interactive terminal
(press `/`). Assets live in `assets/`; deployed by GitHub Pages from `main`.

## Run locally

```bash
python3 -m http.server 8000   # → http://localhost:8000
```
